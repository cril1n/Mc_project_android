# 🍔 Food Delivery App

Una piccola app in React Native per ordinare cibo a domicilio e gestire le consegne. Un progetto semplice creato per fare pratica con lo sviluppo mobile android.

## ✨ Funzionalità
Esplorazione e scelta dei locali da un elenco o dalla mappa.

Tracciamento dell'ordine in corso.

Schermata dedicata alla gestione e alla personalizzazione del profilo.

## 🛠️ Tecnologie
Kotlin 

Jetpack Compose
